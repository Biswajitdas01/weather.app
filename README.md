# SkyCast Weather App

SkyCast Weather App is a simple React.js project that shows current weather information for selected cities.

## Features

- Search weather by city
- Uses React components
- Uses React hooks: useState
- Fetches live weather data from Open-Meteo API
- Shows temperature, wind speed, and weather condition
- Responsive design for mobile and desktop

## Cities Available

- Los Angeles
- New York
- London
- Dhaka
- Tokyo

## How to Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Project Type

React.js Weather App using Vite.
